# `@galactica-net/overcast-private-stablecoin`

Placeholder workspace for the **Aztec (Noir)** token that backs private balances and transfer rules, including hooks for compliant / selective-disclosure flows described in the Overcast docs.

**Reference implementation (upstream):** Aztec’s [`private_token_contract`](https://github.com/AztecProtocol/aztec-packages/tree/next/noir-projects/noir-contracts/contracts/app/private_token_contract) in `aztec-packages` (`next` branch). This package will fork or follow that contract as a baseline.
