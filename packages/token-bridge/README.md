# `@galactica-net/overcast-token-bridge`

Placeholder workspace for **L1 ↔ Aztec** bridging infrastructure so wrapped tokens can be represented and consumed privately on Aztec.

**Reference implementation (upstream):** Aztec’s [`token_bridge_contract`](https://github.com/AztecProtocol/aztec-packages/tree/next/noir-projects/noir-contracts/contracts/app/token_bridge_contract) in `aztec-packages` (`next` branch). This package will adapt or extend those patterns for Overcast.
